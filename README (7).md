## ✨ Features

- 🌑 Dark aesthetic with animated glowing blobs
- ✍️ Elegant Playfair Display + Outfit font pairing
- 🎯 Scroll-triggered reveal animations
- 📊 About section with code-style bio display
- 🛠️ Skills section with clickable tags
- 💼 Projects grid with hover effects
- 📬 Contact section with links
- 📱 Fully responsive on mobile

## 🚀 How to Use

1. Download `index.html`
2. Open in any browser — it works instantly!
3. Edit your name, email, LinkedIn, and project links inside the file

### Things to update:
| What | Find in file | Replace with |
|---|---|---|
| Your name | `Nikita Kalbande` | your real name |
| Email | `YOUR_EMAIL@gmail.com` | your email |
| LinkedIn | `YOUR_LINKEDIN` | your LinkedIn username |
| GitHub links | `NikitaK125` | your GitHub username |

---

## 🌍 Host for Free on GitHub Pages

1. Create a repo named `portfolio`
2. Upload `index.html`
3. Go to **Settings → Pages → Source → main branch**
4. Your site is live at `https://YOUR_USERNAME.github.io/portfolio` 🎉
